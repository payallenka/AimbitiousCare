# Ambitious Care

Mental health platform for construction workers.

## Quick Start

1. Read **[START_HERE.md](./START_HERE.md)** for complete setup instructions
2. Run `npm install`
3. Configure Supabase (see START_HERE.md)
4. Run `npm run dev`

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (Auth, Database, Storage)
- Framer Motion

## Important Files

- `START_HERE.md` - Complete setup guide
- `SUPABASE_SQL_SETUP.sql` - Database schema
- `.env` - Environment variables

That's it!
